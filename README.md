# Abstraction Example

A simple Java application demonstrating the Object-Oriented Programming (OOP) concept of Abstraction using an abstract class.

## Features

- Abstract class (Shape)
- Abstract method
- Method implementation in child class
- Runtime object creation

## Technologies Used

- Java
- IntelliJ IDEA
- Git
- GitHub

## Concepts Covered

- Abstraction
- Abstract Class
- Abstract Method
- Method Overriding
